# Bloc-Notes Electron

Application de bloc-notes faite avec Electron.

## Installation

```
npm install
```

## Lancement

```
npm start
```

## Build

```
npx electron-builder --win
```

## Fonctionnalites

- Ouvrir et sauvegarder des fichiers .txt
- Menu Fichier avec raccourcis (Ctrl+N, Ctrl+O, Ctrl+S)
- Compteur de caracteres en temps reel
- Notification apres sauvegarde
- Bascule theme clair / sombre
- Confirmation avant fermeture si modifications non sauvegardees
